Web Design

Web Design encompasses different skills disciplines in the production and maintence of website. the different areas of web design include web graphic design. user interface design (UI design); authoring, include standarised code and proprietary software; user experience design(UX design); and search engine optimization. often many individuals will work in teams covering different aspects of the design process, although some designers will cover the all. the term "web design" is normally used to describe teh design process relating to the front-end (client side) design of a website include writting markup. web design partially overlaps web engineering in the broader scope of web developmet. web design are expected to have an awarness of usability and be up to date with web accessibility guidelines.
